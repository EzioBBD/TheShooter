项目启动路径：\Assets\_MyGame\_Scenes\Main.unity（出于兼容性考虑，请使用Unity2018打开工程）
可执行文件路径：\Built\Built.exe（由于历史遗留bug，启动配置中Graphics quality请选择Good以上级别）

游戏规则和玩法：
1.玩家两人各持键鼠和手柄对战，击败对方的玩家即获胜
2.左右下角有补给箱可补充子弹和血量
3.问号箱子随机刷出道具，效果见下方道具功能表
4.玩家可在地图中间最上方的控制台处与其交互，水面会上升，玩家碰到立刻死亡
5.弹簧可踩踏弹跳，木箱收到一定伤害后会被破坏

按键功能表：
功能			键鼠		手柄（Xbox）
移动			WASD		左摇杆
跳跃			空格		A
瞄准			鼠标指针		右摇杆
射击			鼠标左键		RB
投掷道具（长按蓄力）	鼠标右键		LB
交互键			E		X
触发遥控炸弹		F		B
菜单/暂停		Esc		Start

道具功能表：
1.火焰弹：强化子弹使其附加灼烧效果
2.寒冰弹：强化子弹使其附加减速效果（可叠加）
3.手雷：投掷一段时间后自动爆炸
4.遥控炸弹：投掷后不会爆炸，可一次性全部触发
